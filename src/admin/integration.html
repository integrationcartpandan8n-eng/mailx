Cadastrar Webhook na CartPanda

Siga os passos abaixo para registrar o webhook na página da API da CartPanda:

Passo a passo

  1. Acesse o painel da CartPanda → Configurações → Integrações → API
  2. Na seção Webhooks, clique em Adicionar Webhook
  3. Cole a URL do webhook copiada no passo anterior
  4. Insira o Shop Slug:
  5. Use o Token ativo para autenticação
  6. Selecione os eventos: order.paid, checkout.abandoned
  7. Clique em Salvar
